◆ STSPIN32G4_init()

STSPIN32G4_StatusTypeDef STSPIN32G4_init ( STSPIN32G4_HandleTypeDef *  hdl)

Initialize the STSPIN32G4 driver.

Precondition
The STSPIN32G4_HandleTypeDef structure must exist and be persistent before this function can be called.
This function must also be called before any other driver APIs.
The driver uses I2C3 interface which must be already initialized.
Initialization example:
STSPIN32G4_HandleTypeDef hdlG4;
STSPIN32G4_StatusTypeDef ret = STSPIN32G4_init(&hdlG4);
