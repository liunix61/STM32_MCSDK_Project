◆ OverVoltageThresholdLow

uint16_t RDivider_Handle_t::OverVoltageThresholdLow

It represents the over voltage protection intervention threshold low level for hysteresis feature when "switch on brake resistor" is used. To be expressed in digital value through formula: hOverVoltageThresholdLow (digital value) = Over Voltage Threshold Low(V) * 65536 / hConversionFactor
