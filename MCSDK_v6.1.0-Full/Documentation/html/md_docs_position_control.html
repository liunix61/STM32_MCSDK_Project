Position Control

Overview

The position control allows the electric drive to move the motor to a specified target mechanical position.

The movement of the motor can be executed in two different ways:

  • In trajectory mode: to achieve a smooth movement, an internal trajectory is calculated according to the given time (considered as a sequence of target positions). This trajectory is composed of three phases:
  1. Acceleration
  2. Constant speed

  3. Deceleration.

    Follow such defined trajectory is mandatory to reach the target position with zero angular speed and therefore avoid oscillations around the settled position.

  • In follow mode: this mode is for instance useful when the trajectory is computed externally by the user.

In both cases, the position control requires:

  • Quadrature encoder (with index signal)
  • Current controller for field oriented control (FOC)
  • Dedicated PID position controller
Position Control Overview

Implementation

The position control method makes use of three controllers as shown in block diagram below.

  • A position controller implemented using a PID (with a proportional, integral and derivative action) executed at Medium Frequency Task (usually 1 kHz).
  • Two current controllers implemented using a PI (with only proportional and integral action) executed at FOC frequency for Id and Iq current regulations.

Both frequencies can be customized by means of the MC Workbench.

The position controller PID influences only the Iq reference (Iqref). Instead, the Id reference (Idref) is managed, as usual, according to the MTPA trajectory.

This scheme is the best approach using an encoder, for the motor position measurement, since it does not require a precise measurement of the motor speed.

Position Control principle

Trajectory mode

In trajectory control mode, a trapezoidal target angular-velocity trajectory is implemented with three time intervals: a constant angular acceleration (from O to B), a constant angular velocity (from B to C), and a constant angular deceleration (from C to D) as shown in blue in Figure below, where l is the duration of the cruise angular velocity interval.

Trajectory With Constant Jerk

To avoid step changes during the angular acceleration interval, which is physically impossible, and to avoid mechanical stress, a constant angular jerk is implemented during the acceleration and deceleration intervals (pink curve).

The angular jerk, also known as jolt, is the rate of change of the angular acceleration with respect to time.

Considering that the deceleration step is symmetrical to the acceleration step, we can assume that the Δθ in acceleration and deceleration steps is equal to \(\theta_3\) given in Table 3:

\(\Delta\theta=\Delta\theta_{EndOfAcceleration}=\Delta\theta_{EndOfDeceleration}=\theta_3\)

The angular variation Δθ in the interval where the angular velocity is constant (cruise interval in Figure 1) and equal to: \(l \timesω_{cruise} \quad \text{where} \quad ω_{cruise}= ω_3 \quad \text{given in Table 3} \)

The total angular variation from the beginning to the end of the movement is expressed in (1):

\(\color{green}{\Delta\theta_{tot}} \color{black}=\Delta\theta_{EndOfAcceleration}+\Delta\theta_{EndOfCruise}+\Delta\theta_{EndOfDeceleration}\)

\(\color{green}{\Delta\theta_{tot}} \color{black}=\theta_3 + l \omega_3 + \theta_3\)

\(\color{green}{\Delta\theta_{tot}} \color{black}=2\theta_3 + l \omega_3=6JA^3+2JA^2l \quad\color{grey}{\text{(1)}}\)

The total duration of the movement is the sum of each interval. It is expressed in (2):

\(\color{blue}{\Delta{t}} \color{black}= 3A + 3A + l \quad\color{grey}{\text{(2)}}\)

To simplify the calculations, it is possible to make the hypothesis that each interval in Figure 3 (acceleration, cruise and deceleration) has the same duration, resulting in \(l= 3A\).

Under this condition, (1) and (2) become:

\(\color{green}{\Delta\theta_{tot}} \color{black}= 12JA^3 \)

\(\color{blue}{\Delta{t}} \color{black}= 9A = l \)

The variables (A, l, J and \(ω_{cruise}\)) listed below are computed according to the two inputs provided by user:

  • ​ \(\color{green}{\Delta\theta_{tot}}\) (total angle movement) and
  • ​ \(\color{blue}{\Delta{t}}\) (movement duration)

The jerk J is used by the trajectory calculator to integrate (step-by-step) the target position.

\(A=\frac{\color{blue}{\Delta{t}}} {9} \)

\(l=\frac{\color{blue}{\Delta{t}}}{3} \)

\(J=\frac{\color{green}{\Delta\theta_{tot}}}{12A^3} \)

\(\omega_{cruise} = 2JA^2 \)

Follow mode

In follow mode, the user can send at fixed rate, different positions according to a required trajectory, and the position control algorithm computes the intermediate points to reach (follow) the target with a smooth movement.

Workbench activation

Once a quadrature encoder is selected in workbench for speed and position sensing, in the Drive Settings window select the Position Control item from the Control mode.
