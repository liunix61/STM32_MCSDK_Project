Detailed Description

Handle structure of a Potentiometer component.

A potentiometer component aims at measuring the voltage across a potentiometer thanks to an ADC.

This structure contains all the information needed for a Potentiometer component instance to work.

See the Linear Potentiometer component documentation for more details.

Field Documentation

◆ LPFilterBandwidth

uint8_t Potentiometer_Handle_t::LPFilterBandwidth

Number of measures used to compute the average value.

This value is set to \(2^{LPFilterBandwidthPOW2}\) by the POT_Init() function.

See LPFilterBandwidthPOW2.

◆ LPFilterBandwidthPOW2

uint8_t Potentiometer_Handle_t::LPFilterBandwidthPOW2

Number of measures used to compute the average value, expressed as a power of 2.

For instance, if the number of measures is 16, the value of this field must be 4 since \(16 = 2^4\).

This field must be set prior to calling POT_Init()

◆ PotMeasArray

uint16_t* Potentiometer_Handle_t::PotMeasArray

Array for storing raw potentiometer measures.

This field must point to a reserved array of uint16_t prior to calling the POT_Init() function. The size of this array must be \(2^{LPFilterBandwidthPOW2}\).

See LPFilterBandwidthPOW2.
