◆ hC3

int16_t STO_CR_Handle_t::hC3

State observer constant C3, it can be computed as F1 * Max application speed [rpm] * Motor B-emf constant [Vllrms/krpm] * sqrt(2)/ (Ls [H] * max measurable current (A) * State observer execution rate [Hz])

◆ hC4

int16_t STO_CR_Handle_t::hC4

◆ hC5
