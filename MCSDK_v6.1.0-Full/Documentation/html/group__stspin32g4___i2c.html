◆ STSPIN32G4_lockReg()

STSPIN32G4_StatusTypeDef STSPIN32G4_lockReg ( STSPIN32G4_HandleTypeDef *  hdl)

Lock protected registers.

Parameters
[in]hdlDriver handler.
Note
With flag STSPIN32G4_I2C_LOCKUSEPARANOID the lock bit of status register will be tested to 1 at the end of locking.
Returns
STSPIN32G4_ERROR in case of failure otherwise STSPIN32G4_OK.

Un-Lock protected registers.

Parameters
