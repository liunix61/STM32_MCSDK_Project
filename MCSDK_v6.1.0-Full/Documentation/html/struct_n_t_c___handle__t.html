◆ hOverTempDeactThreshold

uint16_t NTC_Handle_t::hOverTempDeactThreshold

Temperature threshold below which an active over temperature fault is cleared. This parameter is expressed in u16Celsius through formula: hOverTempDeactThreshold = (V0[V]+dV/dT[V/°C]*(OverTempDeactThresh[°C] - T0[°C]))* 65536 / MCU supply voltage

◆ hOverTempThreshold

uint16_t NTC_Handle_t::hOverTempThreshold

Represents the over voltage protection intervention threshold. This parameter is expressed in u16Celsius through formula: hOverTempThreshold = (V0[V]+dV/dT[V/°C]*(OverTempThreshold[°C] - T0[°C]))* 65536 / MCU supply voltage
