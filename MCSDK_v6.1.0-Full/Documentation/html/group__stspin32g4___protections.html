◆ STSPIN32G4_setInterlocking()

STSPIN32G4_StatusTypeDef STSPIN32G4_setInterlocking ( STSPIN32G4_HandleTypeDef *  hdl,
bool  enabled 
)

Configures the interlocking.

The device integrates interlocking features that prevents the power stage from accidental cross conduction.
HS and LS MOSFET of same channel cannot be turned on simultaneously when interlocking is active.

Parameters
[in]hdlDriver handler
[in]enabledIf 1 the interlocking is enabled.
If 0 the interlocking is disabled.
Warning
Disable the interlocking is potentially desruptive for power stage.
Returns
STSPIN32G4_ERROR in case of failure otherwise STSPIN32G4_OK.

◆ STSPIN32G4_setMinimumDeadTime()

STSPIN32G4_StatusTypeDef STSPIN32G4_setMinimumDeadTime ( STSPIN32G4_HandleTypeDef *  hdl,
bool  enabled 
)

Configures the minimum deadtime.

The device integrates a minimum deadtime features. When enabled a minimum delay is imposed between the turn-off of one MOSFET and the turn-on of the complementary one.

Parameters
[in]hdlDriver handler
[in]enabledIf 1 the minimum deadtime is enabled
If 0 the the minimum deadtime is disabled.
The minimum deadtime is not added to the one imposed by the MCU
Returns
STSPIN32G4_ERROR in case of failure otherwise STSPIN32G4_OK.
