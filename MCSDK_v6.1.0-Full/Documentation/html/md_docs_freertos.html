MCSDK: FreeRTOS documentation

Overview

The ST Motor Control SDK has been developed to support two Motor Control tasks scheduled by FreeRTOS:

  • the medium frequency tasks
  • the safety task

The chosen memory management strategy is heap 4 ( default one proposed by STM32CubeMX, for more information visit FreeRTOS website ).
It is advised to select a timebase source other than the Systick when FreeRTOS is enabled: one of the available timers of the MCU.

FreeRTOS activation

During the project configuration the following steps needs to be done to activate FreeRTOS:

  • navigate to Drive Management and select FreeRTOS
  • enable freeRTOS by clicking the checkbox
  • according to the application, the MC IRQ may use freeRTOS functions. In that case enable it by clicking on the checkbox
  • select one of the available timers for the timebase source

Performance

The measurements have been done with the following configuration:

  • Single drive
  • Three shunts current sensing topology
  • Sensorless
  • HAL drivers
