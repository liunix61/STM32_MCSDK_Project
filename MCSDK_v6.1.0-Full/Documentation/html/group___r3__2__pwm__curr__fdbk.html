Detailed Description

3-Shunt, 2 ADCs, PWM & Current Feedback implementation for F30X, F4XX, F7XX, G4XX, H7XX and L4XX MCUs

This component is used in applications based on STM32 MCUs, using a three shunt resistors current sensing topology and 2 ADCs peripherals to acquire the current values.
