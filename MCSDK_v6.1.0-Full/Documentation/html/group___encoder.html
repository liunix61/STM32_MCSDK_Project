Detailed Description

Quadrature Encoder based Speed & Position Feedback implementation.

This component is used in applications controlling a motor equipped with a quadrature encoder.

This component uses the output of a quadrature encoder to provide a measure of the speed and the motor rotor position.

More detail in Encoder Speed & Position Feedback module.

◆ ENC_CalcAvrgMecSpeedUnit()

__weak bool ENC_CalcAvrgMecSpeedUnit ( ENCODER_Handle_t *  pHandle,
int16_t *  pMecSpeedUnit 
)

This method must be called with the periodicity defined by parameter SpeedSamplingFreqUnit. The method generates a capture event on a channel, computes and stores average mechanical speed expressed in the unit defined by SPEED_UNIT (on the basis of the buffer filled by Medium Frequency Task), computes and stores average mechanical acceleration expressed in SPEED_UNIT/SpeedSamplingFreq, computes and stores the instantaneous electrical speed in Digit Per control Period unit dpp, updates the index of the speed buffer, then checks, stores and returns the reliability state of the sensor.

Parameters
pHandlehandler of the current instance of the encoder component
pMecSpeedUnitpointer used to return the rotor average mechanical speed expressed in the unit defined by SPEED_UNIT
Return values
true= sensor information is reliable. false = sensor information is not reliable
