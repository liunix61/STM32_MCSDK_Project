Detailed Description

PWM generation implementation for Six-Step drive with 3 PWM duty cycles.

This implementation only drives the high sides of the bridges. The low sides are automatically driven by the gate driver that inserts the dead time.

Three additional signals are generated to enable or disable the PWM output of each phase.
