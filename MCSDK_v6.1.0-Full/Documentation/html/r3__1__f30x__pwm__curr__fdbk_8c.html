Detailed Description

This file provides firmware functions that implement current sensor class to be stantiated when the three shunts current sensing topology is used.

Author
Motor Control SDK Team, ST Microelectronics

It is specifically designed for STM32F30X microcontrollers and implements the successive sampling of two motor current using only one ADC.

  • MCU peripheral and handle initialization function
  • three shunt current sensing
  • space vector modulation function
  • ADC sampling function
