Detailed Description

Handle structure of a Speed Potentiometer component.

A Speed Potentiometer component reads the value set on a potentiometer and uses it to set the rotation speed reference of a motor. To get potentiometer values, an instance of a Potentiometer component is used.

This structure contains all the information needed for a Speed Potentiometer component instance to work.

See the Speed Potentiometer component documentation for more details.
