◆ EmergencyStop

FunctionalState R3_2_Params_t::EmergencyStop

◆ FreqRatio

uint8_t R3_2_Params_t::FreqRatio

Used in case of dual MC to synchronize TIM1 and TIM8. It has effect only on the second instanced object and must be equal to the ratio between the two PWM frequencies (higher/lower). Supported values are 1, 2 or 3

◆ hDeadTime

uint16_t R3_2_Params_t::hDeadTime

Dead time in number of TIM clock cycles. If CHxN are enabled, it must contain the dead time to be generated by the microcontroller, otherwise it expresses the maximum dead time generated by driving network. Specific to F4XX and F7XX.

◆ hTafter

uint16_t R3_2_Params_t::hTafter

Sum of dead time plus max value between rise time and noise time express in number of TIM clocks. Specific to F4XX and F7XX.

◆ IsHigherFreqTim

uint8_t R3_2_Params_t::IsHigherFreqTim

When FreqRatio is greater than 1 this param is used to indicate if this instance is the one with the highest frequency. Allowed values are: HIGHER_FREQ or LOWER_FREQ
