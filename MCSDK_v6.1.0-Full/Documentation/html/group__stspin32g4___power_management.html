Enable or Disable the LDO linear regulator.

Parameters
[in]hdlDriver handler
[in]enabledIf 1 the 3.3V supply is internally generated via the LDO linear regulator.
If 0 the LDO regulator is disabled and the 3.3V supply have to be provided externally.

The device will perform a reset if the LDO is disabled without providing externally the 3.3V supply.

STSPIN32G4_ERROR in case of failure otherwise STSPIN32G4_OK.

◆ STSPIN32G4_standby()

STSPIN32G4_StatusTypeDef STSPIN32G4_standby ( STSPIN32G4_HandleTypeDef *  hdl,
bool  enableStbyReg 
)

Requests the device to enter standby mode.

In standby mode the device turns off most of its internal circuitry with inclusion of Buck converter, LDO linear regulator and gate drivers.
In standby the overall current consumption is reduced to ~15uA and the MCU can be supplied via dedicated low quiescent linear regulator from main supply voltage.

Parameters
[in]hdlDriver handler
[in]enableStbyRegIf 1 the internal low quiescent linear regulator will be activated when entering standby mode
Warning
Overall consumption from 3.3V supply should be reduced as much as possible before entering standby.
In case the consumption exceeds 5mA the device will automatically exit from standby performing a reset.
Disable all MCU peripherals and reduce system clock frequency before calling function STSPIN32G4_standby().
If flag STSPIN32G4_HSI16 is used, the driver will automatically reduce system clock frequency.
Returns
STSPIN32G4_ERROR in case of failure otherwise STSPIN32G4_OK.
